# Agentic AI Lab 2026: Autonomous Allocation & Governance

This laboratory serves as the technical implementation of my PhD research in **Business Sciences**, focusing on the intersection of **Agentic AI**, **Mechanism Design**, and **Economic Incentives**.

## Research Focus
Testing the viability of:
* AAM (Autonomous Allocation Mechanism):** Decentralized resource distribution in cloud environments
* GMD (Governance as Mechanism Design):** Dynamic policy enforcement for multi-agent systems
* AMD (Adversarial Mechanism Design):** Testing system robustness against strategic/malicious agent behavior

## Tech Stack
* Orchestration:** LangGraph / FastAPI
* Models:** Gemini 1.5 Flash & Gemma 2 (via Ollama/GKE)
* Infrastructure:** Google Cloud Platform (Cloud Run & GKE)
* Visualization:** Next.js + D3.js (Real-time WebSockets)
